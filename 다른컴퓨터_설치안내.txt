==========================================
  매출 분석 프로그램 — 다른 컴퓨터 설치 안내
  (EXE 배포 · Python 설치 불필요)
==========================================

■ 준비물
  - Windows 10/11 PC
  - 이카운트 등 ERP 판매 CSV (품목·거래처 통합 1개 + 구매 선택)


■ 개발 PC에서 EXE 만들기 (수정 후 · 인터넷 필요)

  1. Python 3.10 이상 설치 (PATH 체크)
  2. "앱빌드.bat" 더블클릭 (5~10분)
  3. 완료 후 "배포용\매출분석" 폴더가 생성됨


■ 다른 PC에 옮기기

  [방법 A] 분석·저장 데이터까지 그대로 옮기기 (권장)

  1. 개발 PC에서 복사할 것
     · 배포용\매출분석\          ← 프로그램 전체
     · (선택) 저장데이터\       ← exe 옆에 이미 있으면 폴더 통째로 포함

  2. USB 등으로 다른 PC에 붙여넣기 (예: D:\매출분석)

  3. 매출분석.exe 실행
  4. "이전 CSV·월별 데이터를 복원했습니다" 확인
  5. ★ [분석 실행] 다시 누르지 말 것 ★

  [방법 B] 프로그램만 새로 설치

  1. "배포용\매출분석" 폴더만 복사
  2. 매출분석.exe 실행
  3. (1) 판매 CSV + (2) 구매 CSV(선택) 불러오기 → [분석 실행]


■ CSV 형식 (2026년 기준)

  판매 1개 파일에 포함:
    품목코드, 품목명(규격), 일별, 거래처코드, 수량, 합계 등

  구매 (선택):
    거래처별 구매현황 CSV


■ 저장데이터 폴더 (exe 옆)

  months.json      월별 분석 결과
  sales.csv        판매 CSV 백업
  purchase.csv     구매 CSV 백업
  item_map.json    품목 분류표
  cust_map.json    거래처 분류표


■ 개발 PC에서 수정 후 다시 배포

  1. 매출분석.html 등 수정
  2. "앱빌드.bat" 다시 실행
  3. 새 "배포용\매출분석"을 다른 PC에 덮어쓰기
  4. 기존 "저장데이터" 폴더는 삭제하지 말고 유지
