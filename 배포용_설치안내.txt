==========================================
  매출 분석 프로그램 — 설치 및 사용 안내
  (다른 PC · USB 복사용)
==========================================

■ 필요 환경
  · Windows 10 / 11 (64비트)
  · Python 설치 불필요 (매출분석.exe 포함)
  · 이카운트 등 ERP에서 받은 CSV


■ 처음 설치 (3단계)

  1. 이 폴더 전체를 다른 PC에 복사합니다.
     예) D:\매출분석

  2. 「매출분석.exe」 또는 「실행.bat」을 더블클릭합니다.
     ※ 매출분석.html 만 더블클릭하면 저장·복원이 되지 않습니다.

  3. CSV를 불러온 뒤 [분석 실행]을 누릅니다.


■ CSV 불러오기

  (1) 품목·거래처별 판매현황  [필수]
      · 품목코드, 거래처코드, 일별(또는 일자), 수량, 합계가
        한 행에 있는 통합 CSV 1개
      · 예) 품목별·거래처별 매출자료, 전표별품목별매출자료

  (2) 거래처별 구매현황  [선택]
      · 지역분류(경상남도 / 경상남도외) 보고서에 필요할 때만

  · 이미 불러온 파일이 있으면 덮어쓰기 여부를 묻습니다.
  · 판매 CSV만 바꿔도 구매 CSV는 유지됩니다.
  · [지우기]는 확인 후에만 삭제됩니다.


■ 메뉴 구성

  1. 연간월별 데이터   — CSV 분석, 월별 저장, 결과 표
  2. 기간 비교분석 보고서 — 저장된 월 데이터로 비교 보고서
  3. 분류코드 관리     — 품목·거래처 분류코드, 직납 브랜드


■ 데이터 저장 위치 (exe와 같은 폴더)

  저장데이터\     월별 분석·CSV 백업·분류표
  다운로드\       엑셀 다운로드 결과

  주요 파일:
    months.json    월별 분석 결과
    sales.csv      판매 CSV 백업
    purchase.csv   구매 CSV 백업 (있을 때)
    item_map.json  품목 분류표
    cust_map.json  거래처 분류표


■ 다른 PC로 데이터 옮기기

  1. 기존 PC의 이 폴더 안 「저장데이터」폴더를 통째로 복사
  2. 새 PC의 매출분석.exe 가 있는 폴더에 붙여넣기
  3. exe 실행 → "이전 CSV·월별 데이터를 복원했습니다" 확인
  4. ★ [분석 실행] 다시 누르지 말고 ★
     저장된 월을 선택해 보고서만 생성


■ 문제 해결

  실행 안 됨 / 백신 차단
    → Windows Defender 예외 추가
    → "추가 정보" → "실행"

  저장 데이터가 안 보임
    → 반드시 매출분석.exe 로 실행
    → 저장데이터 폴더가 exe 옆에 있는지 확인

  1월 금액만 비정상적으로 큼
    → 여러 달이 한 달로 합쳐 저장된 경우
    → 저장데이터 삭제 후 CSV 다시 [분석 실행]
    → CSV에 「일별」또는「월별」열 확인

  한글 깨짐
    → CSV를 UTF-8로 저장 (이카운트 기본값)
