매출분석 프로그램 — 설치 및 실행 안내
================================

1. 압축 파일(매출분석프로그램_배포.zip)을 원하는 위치에 풀어 주세요.
2. 풀린 폴더 안의 「매출분석프로그램.exe」를 더블클릭하세요.
3. 잠시 후 웹 브라우저가 자동으로 열리며 프로그램이 시작됩니다.

※ Python 설치는 필요 없습니다.
※ 프로그램을 종료하려면 exe 실행 시 뜨는 검은 창(콘솔)을 닫으세요.
※ 데이터(매출자료, 분류코드)는 exe와 같은 폴더에 저장됩니다.
   폴더를 통째로 복사하면 다른 PC에서도 데이터를 그대로 사용할 수 있습니다.
